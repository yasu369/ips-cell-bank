# Double-K iPS Cell Bank

Independent static site for the Double-K iPS Cell Bank concept.

## Local Preview

```powershell
node serve.js
```

Open `http://127.0.0.1:4173/` from this project directory.

To preview without conflicting with the existing site in the parent project:

```powershell
node serve-4175.js
```

Open `http://127.0.0.1:4175/`.
